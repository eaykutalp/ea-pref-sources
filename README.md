# Google Preferred Sources Helper (WordPress Plugin)

Bu WordPress eklentisi, blog yazılarındaki ilk paragrafın hemen üstüne ziyaretçileri sitenizi Google'da **"Tercih Edilen Kaynak" (Preferred Source)** olarak eklemeye davet eden modern, responsive ve şık bir kart enjekte eder. 

Böylece sadık okuyucularınızın Google Arama sonuçlarında, "En Çok Okunan Haberler" bölümünde ve Google'ın AI Mode / AI Overviews yapay zekâ özetlerinde sitenizin içeriklerini öncelikli olarak görmelerini kolaylaştırır.

## 🌟 Özellikler

- **Vanilla CSS (No-Tailwind):** Harici hiçbir CSS kütüphanesine veya Tailwind'e bağımlı değildir. Saf CSS ile sitenizi ağırlaştırmadan çalışır.
- **Kişisel Marka Tasarım Sistemi:** Emre Aykutalp marka kimliği renklerine (#1A2732, #6D7683, #F7F8FA, #E5E5E5), yazı boyutlarına ve modern Neue Galano yazı tipi ailesine göre tasarlanmıştır.
- **Dinamik İçerik Enjeksiyonu:** Yazı içeriğindeki ilk `<p>` (paragraf) etiketini otomatik tespit ederek kartı tam üzerine yerleştirir.
- **Kolay Yönetim Paneli:** WordPress yönetim panelinde **Ayarlar > Google Preferred Source** sayfası altından tüm metinleri ve hedef alan adını kod yazmadan güncelleyebilirsiniz.
- **Tam Responsive Tasarım:** Masaüstü cihazlarda yan yana, mobil cihazlarda ise dikey yığılmalı ve tam genişlikte buton olacak şekilde esnektir.

## 🛠️ Kurulum

1. Bu depoyu indirin ve klasör adının `ea-pref-sources` olduğundan emin olun.
2. Klasörü `.zip` formatında sıkıştırın (örn: `ea-pref-sources.zip`).
3. WordPress Admin Paneline gidin: **Eklentiler > Yeni Ekle > Eklenti Yükle**.
4. Sıkıştırdığınız `.zip` dosyasını seçin, yükleyin ve **Eklentiyi Etkinleştir** butonuna tıklayın.

## ⚙️ Yapılandırma

Eklentiyi etkinleştirdikten sonra WordPress yönetim panelinizde **Ayarlar > Google Preferred Source** adında yeni bir menü görünecektir. Buradan şu ayarları özelleştirebilirsiniz:

- **Hedef Alan Adı (Domain):** Google Preferred Source deep link yapısında kullanılacak alan adınız (Örn: `emreaykutalp.com`).
- **Kart Başlığı:** Ziyaretçilerin göreceği ana başlık.
- **Kart Açıklaması:** Butonun ne işe yaradığını açıklayan alt metin.
- **Buton Metni:** Eyleme çağrı (CTA) butonunun üzerindeki metin.

## 🖥️ Arayüz Görünümü

### Kullanıcı Arayüzü (Blog Sayfaları)
*Blog yazılarınızın başında, ilk paragrafın hemen üstünde şu şekilde görünür:*

> **Google'da Tercih Edilen Kaynaklarına Ekle**
> Google aramalarında ve yapay zekâ özetlerinde bu sitenin yazılarını öncelikli görün.
> [ Kaynağı Ekle -> ]

### Yönetim Paneli (WordPress Admin)
*Ayarlarınızı düzenleyebileceğiniz sade panel:*

- **Ayarlar > Google Preferred Source** adımlarından kolayca ulaşılabilir.
